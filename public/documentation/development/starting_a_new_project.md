# Starting a new project

## From Github:

1.) Visit repo [ModelRails App](https://github.com/dschmura/modelrails_app)
2.) Click "Use this template" button
3.) Name your new repo and click "Create repository from template"
4.) Clone your new repo to your local machine ```git clone <your repo url>```
5.) ```cd <your repo name>```
6.) run setup script ```./bin/setup```    


## From Local Machine:
1.) Clone this repo to your local machine ```git clone https://github.com/dschmura/modelrails_app <your repo name>```
2.) ```cd <your repo name>```
3.) run setup script ```./bin/setup```
